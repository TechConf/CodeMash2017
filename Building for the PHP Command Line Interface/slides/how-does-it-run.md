### How does it run?

Two ways to invoke PHP on the CLI:

```sh
# 1. Pass the file to the php binary
$ php my-command.php
```
<!-- .element: class="fragment" -->

```sh
# 2. Execute the file directly if using the php shebang.
$ chmod +x my-command.php
$ ./my-command.php
```
